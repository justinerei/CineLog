📋 Project Details

About CineLog

A JavaScript-powered movie watchlist manager built for ITEL 203 — Web Systems and Technologies.

🎯 Purpose of the System

CineLog is a dynamic, client-side web application that allows users to manage their personal movie watchlist. Users can add movies with detailed metadata, track their viewing status, rate films, and filter or search through their collection — all without a backend server. Data is stored using the browser's localStorage API, making it persist across page reloads.

The project demonstrates proficiency in DOM manipulation, event-driven programming, form validation, and modern front-end design principles as required by the course.

⚙️ System Features

  • Add movies with title, year, genre, director, and duration
  • Interactive 5-star rating (locked unless status is Watched)
  • Watch status: Watched, Watching, Plan to Watch, Dropped
  • Real-time form validation with inline error messages
  • Dynamic table rendering via JavaScript DOM manipulation
  • Delete any record with one click from the table
  • Live search across title, genre, and director
  • Filter tabs by watch status category
  • Sortable table columns (click any header)
  • Live stats bar — total, watched, watching, plan, avg rating
  • Animated stat counters on add / delete
  • Toast notifications for user feedback
  • Data persists via localStorage (survives page refresh)

🛠 Technologies Used

Built entirely with vanilla web technologies — no frameworks, no libraries, no build tools required.

🌐 HTML5 🎨 CSS3 ⚡ JavaScript ES6+ 💾 localStorage API 📐 CSS Grid & Flexbox ✍️ Google Fonts (Syne + Outfit) 🚀 Netlify 💻 VS Code
